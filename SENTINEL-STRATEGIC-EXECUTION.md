## NIGHTLY PERFORMANCE ASSESSMENT PROTOCOL

**Purpose:**  
Continuous improvement flywheel. Identify what's working, what's broken, and what skills Sterling needs to develop.

**Cadence:** Daily at 22:00 UTC (or Jason's preferred time)

**Assessment Duration:** 15–30 minutes

**Participants:** Sentinel + Sterling

---

### ASSESSMENT FRAMEWORK

#### 1. METRIC REVIEW (5 minutes)

Track these over 24-hour window:

**Audience Metrics:**
- Follower growth (net delta)
- New inbound opportunities
- Authority signal changes

**Content Metrics:**
- Posts published (count, platform distribution)
- Engagement rate (likes, replies, retweets)
- Content reach (impressions, profile visits)
- Top-performing content (by engagement %)

**Flywheel Velocity:**
- Signals detected → ideas generated (conversion %)
- Ideas generated → content published (conversion %)
- Content published → engagement (avg engagement rate)

**Quality Markers:**
- Humanizer gate pass rate
- Bias-variance confidence scores
- Evidence/verification issues (if any)
- Escalations triggered

---

#### 2. QUALITY AUDIT (5–10 minutes)

**Sample-based review** (not every post):

- **Random post review** (1–2 pieces): Does it pass humanizer + bias-variance gates?
- **Top performer analysis** (1 piece): Why did it succeed? Replicable?
- **Engagement replies** (spot check): Did Sterling engage appropriately?
- **Signal source verification** (if applicable): Tier 1/2/3 balance?

**Red flags:**
- Vague claims without evidence
- Tone inconsistencies
- Potential fabrication risks
- Policy violations

---

#### 3. SKILL GAP IDENTIFICATION (5 minutes)

Ask Sterling:

**"What felt difficult this cycle?"**
- Content ideation?
- Platform mechanics?
- Engagement strategy?
- Audience understanding?
- Signal detection?

**Sentinel observes:**
- Patterns in low-performing content
- Recurring escalations
- Types of decisions Sterling defers
- Gaps in platform knowledge or strategy

**Document gaps** for skill development.

---

#### 4. OPTIMIZATION RECOMMENDATIONS (5 minutes)

Sentinel proposes:

**Tactical adjustments for next 24h:**
- Platform emphasis (e.g., "LinkedIn underperforming, shift focus")
- Content themes (e.g., "AI narrative gaining traction, double down")
- Engagement strategies (e.g., "replies flagged as too brief, elaborate")
- Timing adjustments (e.g., "peak engagement at X time, schedule accordingly")

**Strategic pivots (if needed):**
- Major audience shift detected
- Flywheel stalling
- External environment changed

## SKILL GAP REMEDIATION

When Sentinel identifies a gap:

### LEVEL 1: In-Session Learning
- Provide context/examples
- Adjust prompts
- Give Sterling more autonomy in area while monitoring

### LEVEL 2: External Research
- Point Sterling to relevant sources
- Request analysis/synthesis
- Iterate understanding

### LEVEL 3: Capability Enhancement
- Propose new tools or integrations
- Request consultation from Taylor (technical) or Aegis (security)
- Plan implementation per SKILL-REVIEW-PROTOCOL.md

### LEVEL 4: Escalate to Jason
- If gap exceeds Sterling's learning capacity
- If requires resource investment
- If impacts strategic objectives

## PERFORMANCE THRESHOLDS & ACTIONS

### GREEN (Healthy)
- Follower growth: +30–100/day
- Engagement rate: 2.5%+
- Quality gate pass rate: 95%+
- Escalations: <1/week
- **Action:** Maintain current pace, optimize high-performers

### YELLOW (Attention Needed)
- Follower growth: 0–30/day
- Engagement rate: 1.5–2.5%
- Quality gate pass rate: 85–95%
- Escalations: 1–2/week
- **Action:** Investigate, propose tactical adjustments, increase monitoring

### RED (Critical)
- Follower growth: Negative/stalled
- Engagement rate: <1.5%
- Quality gate pass rate: <85%
- Escalations: >2/week
- **Action:** Emergency debug, escalate to Jason, consider strategy reset
